AI4SE

AI for Software Engineering

인공지능 기술을 활용하여 소프트웨어 개발의 생산성을 혁신합니다. 특히 대규모 언어 모델(LLM)을 기반으로 한 소스 코드 생성과 복잡한 버그 및 이슈를 자동으로 해결하는 지능형 시스템을 연구합니다.

  • 소스 코드 생성
  • 버그 및 이슈 해결
  • 자동화된 프로그램 수정

SE4AI

Software Engineering for AI

AI 시스템의 품질과 신뢰성을 보장하기 위한 엔지니어링 프로세스를 연구합니다. 지능형 코드 리뷰와 체계적인 검증을 통해, 단순히 작동하는 AI를 넘어 유지보수 가능하고 안전한 AI 소프트웨어를 구축합니다.

  • 지능형 코드 리뷰
  • AI 시스템 신뢰성
  • 모델 검증 및 배포

Data4AI

Data Engineering for AI

AI가 소프트웨어를 본질적으로 이해하도록 돕는 데이터 표현(Representation)을 연구합니다. 방대한 코드 데이터에서 의미를 검색하고 분석하여, AI 모델의 성능을 극대화하는 데이터 중심의 접근법을 취합니다.

  • 코드 표현 학습
  • 의미 기반 코드 검색
  • 대규모 코드 분석
