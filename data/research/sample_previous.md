# IoT 디바이스 보안 분석

이 연구는 다양한 IoT 기기 펌웨어의 보안 취약점을 자동으로 분석하는 프레임워크를 개발했습니다.
동적 분석과 정적 분석을 결합하여 기존 도구보다 높은 탐지율을 달성했습니다.

## 성과
- 주요 컨퍼런스 논문 발표
- 오픈소스 도구 공개

# IoT Device Security Analysis

This research developed a framework for automatically analyzing security vulnerabilities in various IoT device firmwares.
By combining dynamic and static analysis, we achieved a higher detection rate than existing tools.
